# Telco Customer Churn Analysis

## Project Overview

This project focuses on analyzing customer churn behavior in a telecommunications company using SQL Server and Power BI.

The objective of this analysis is to identify the main factors affecting customer churn, uncover business insights, and provide actionable recommendations to improve customer retention and reduce revenue loss.

The project follows a complete end-to-end data analytics workflow including:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Business Analysis
* KPI Development
* Dashboard Design
* Business Recommendations

# Key Insights

* Month-to-month customers showed the highest churn rate.
* High monthly charges increased churn probability.
* Poor customer support experience contributed significantly to churn.
* Pricing dissatisfaction was one of the top churn reasons.
* Revenue loss due to churn represented a significant business risk.

---

# Business Recommendations

* Introduce flexible pricing plans.
* Improve customer support response time.
* Launch customer loyalty and retention programs.
* Encourage long-term contracts through discounts and offers.
* Develop targeted retention campaigns for high-risk customers.
